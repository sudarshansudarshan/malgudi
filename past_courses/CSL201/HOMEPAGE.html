Lab 03

13th August,2014

  1. Write a function partition(a,p,q) where a is the input list and p,q specify the range of a that needs to be partitioned. Return the alpha value where alpha gives the position of pivot elemnt in the sorted list.
  2. Use the partition function to write the code of quick sort

Lab 04

26th August,2014

  1. Write a program to sort a list using Ternary heap sort and compare it with heap sort

    2. Format: ternary_heap(A) that sorts A assuming each node in the heap has 3 children. heap(A) that sorts A assuming each node has two children. compare(A) calls heap sort and ternary heap sort on same input A and prints the difference in the amout of work done

  2. Write a program that simulates Hospital token generator.

    3. The person with the highest criticality needs to treated first. Higher the criticality of a patient, lower the token number generated (will be treated first as compared with the person who has lower criticality rate). The crticality of a patient is dynamic(can be increased). The queue size also changes dynamically.

  3. Write a program that traverses a NxN matrix in a Spiral fashion as given in the test T1.

    4. Format: Function spiral (A) accepts a N*N matrix where n is even and prints its elements in a spiral fashion
